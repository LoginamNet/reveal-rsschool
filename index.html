DEEPCLONE

What if our sheep is a function?

JavaScript does not contain separate functions for deep copying arrays or objects. 

									const arr = [1, [2], [3, [4]], 5];
									const copyArr = JSON.parse(JSON.stringify(arr));

									console.log(copyArr);
									(4) [1, Array(1), Array(2), 5]
But wait, JSON.parse/stringify can do it! Right?

DEEPCLONE

Not realy... 

									const arr = [
										undefined,
										function() { console.log('aaa') },
										Symbol("foo")
									]
									const copyArr = JSON.parse(JSON.stringify(arr));

									console.log(copyArr);
									(3) [null, null, null]
Using this method, the source object must be JSON-safe - we do not use Date, undefined, Infinity, functions, regexps, maps, sets, or other complex types within our array/object.

DEEPCLONE

deepClone work with all types, function and Symbol are copied by reference. 

									import cloneDeep from 'lodash.clonedeep'

									const arr = [
										undefined,
										function() { console.log('aaa') },
										Symbol("foo")
									]
									const copyArr = cloneDeep(arr);

									console.log(copyArr);
									(3) [undefined, ƒ, Symbol(foo)]

Memoize

Memoize lets you cache the results of a function for the same arguments. 

									import memoize from "lodash.memoize";

									function add(a, b) {
										console.log("Add called with ", a, b);
										return a + b;
									}
									  
									const adder = memoize(add);
But memoize uses the first argument to create the key to the cache. If the method you want to memoize takes more than one argument, the additional ones would get ignored.

Memoize

To fix this problem, you can pass a resolver to memoize. 

									import memoize from "lodash.memoize";
									import values from "lodash.values";

									function add(a, b) {
										console.log("Add called with ", a, b);
										return a + b;
									}
									  
									const adder = memoize(add, (...args) => values(args).join("_"));
NOTE:values is the Lodash values method. The resolver takes all the arguments and concatenates them to create a unique string. Now, anytime that same unique set of arguments is used, memoize will return the cached result.

Debounce

Debounce function returns a debounced function that when called will execute a function after X milliseconds pass since its last execution. 

									import debounce from 'lodash/debounce';

									const debounced = debounce(() => {
										console.log('Debounced after 500ms!');
									}, 500);

									window.addEventListener('resize', debounced);
Debouncing is best suited when the result of the last event (for example, when resizing a window) matters to the end user.

Throttle

Throttle creates a throttled function that only invokes the passed function at most once per every interval, specified in milliseconds. 

									import throttle from 'lodash/throttle';

								const throttled = throttle(() => {
									console.log('Throttled after 50ms!');
								}, 50);

								window.addEventListener('resize', throttled);
Trottling is most effective when the input data for a function call does not matter or is the same every time (for example, the scroll event).
