Why should i use it?

Know for yourself. Use the experience of others

If you want to ride a bike, you don't need to invent it. You need to know how to use it and what do you want to achive with your actions.
Lodash can't teach you to "ride", but it can make the trip to the goal more comfortable and faster.

Still actual?

A lot can be done with vanilla js these days...but

With the introduction of the ES6 standard, many Lodash methods are now available out of the box. 

											import { last } from 'lodash';
	
											last(['a', 'b', 'c']);
											//=> 'c'
										

											const arr = ['a', 'b', 'c'];
	
											arr[arr.length - 1];
											//=> 'c'
But You can import the individual functions where needed instead of the full library if the functionality isn't easy to achieve in vanilla js
